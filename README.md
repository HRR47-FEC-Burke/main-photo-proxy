## Usage

> Example URL: http://localhost:4000/item/25

### Installing Dependencies

```sh
npm install
```

### Node Express Server

```sh
npm start
```

## Requirements

- Node.js v14.3.0
  - https://nodejs.org/

## Deployment

- Run $ `docker-compose up -d` to start running the service on port 80
